# Ask AI – YouTube AI Tutor 🚀

An intelligent Chrome extension that transforms YouTube into an interactive learning platform by allowing users to ask AI-powered questions directly from video context.

---

## 🔥 Features

- 🎯 Ask questions directly while watching YouTube videos  
- 🧠 Context-aware AI responses using transcript data  
- ⚡ Real-time answers without leaving the video  
- 🔁 Smart fallback when transcripts are unavailable  
- 🎥 Enhances learning without breaking flow  

---

## 🧠 How It Works

1. Extracts transcript from the YouTube video  
2. Captures the current timestamp  
3. Selects relevant context around that moment  
4. Sends structured prompt to AI (Gemini API)  
5. Returns accurate, simplified explanation  

---

## 🛠️ Tech Stack

- **Frontend:** Chrome Extension (JavaScript, HTML, CSS)  
- **Backend:** Node.js / Python (API handling)  
- **AI Model:** Gemini API  
- **Data Source:** YouTube Transcript API
